# stackmux.github.io

GitHub Pages landing site for StackMux.

## Purpose

This repository is the public-facing site for the StackMux project.

The StackMux CLI source repository is:

`github.com/stackmux/stackmux`

Active local development may happen in a sibling worktree folder.

## Local preview

Because the site is a single static HTML file, a simple local preview is enough:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

The landing page content in [index.html](/home/jpasichnyk/projects/stackmux.github.io/index.html) is based on the current reviewed StackMux README in the main source repository.
